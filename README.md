# Hu Tao Inspired Developer Portfolio

A highly interactive, and aesthetic developer portfolio website inspired by Hu Tao from Genshin Impact. Built to be a modern React application utilizing Vite, TailwindCSS, and Framer Motion, delivering premium interactive features and smooth animations.

## 🚀 Features

- **Thematic Aesthetic**: Custom dark mode with deep reds, blacks, and golds inspired by Hu Tao's color palette.
- **Interactive Animations**: Powered by Framer Motion for smooth page transitions, element reveals, and engaging micro-interactions.
- **Dynamic Components**:
  - 3D tilt-effect cards for projects or skills
  - Custom thematic cursor
  - Dynamic coding-vibe background
- **Responsive Design**: Fully responsive layout that looks great on mobile, tablet, and desktop devices.
- **Modern Tech Stack**: Fast and optimized using Vite, React 18, and TailwindCSS.

## 🛠️ Tech Stack

- [React 18](https://react.dev/) - UI Library
- [Vite](https://vitejs.dev/) - Build Tool & Development Server
- [TailwindCSS](https://tailwindcss.com/) - Utility-first CSS framework for styling
- [Framer Motion](https://www.framer.com/motion/) - Animation library for React
- [Lucide React](https://lucide.dev/) - Beautiful & consistent icons
- [clsx](https://github.com/lukeed/clsx) & [tailwind-merge](https://github.com/dcastil/tailwind-merge) - Utility for constructing CSS class strings

## 🏃‍♂️ Getting Started

### Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository (or download the source code):
   ```bash
   git clone <repository-url>
   cd portof
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` to view the portfolio.

## 📦 Build for Production

To build the project for production, run:

```bash
npm run build
```

This will generate a `dist` directory with the compiled, minified, and optimized assets ready to be deployed. You can preview the production build locally by running:

```bash
npm run preview
```

## 📂 Project Structure

- `src/` - Application source code
  - `components/` - Reusable React components (Navbar, About, Gallery, Skills, Contact, etc.)
  - `App.jsx` - Main application component
  - `index.css` - Global styles and Tailwind configuration
- `public/` - Static assets served directly at the root (e.g., images)
- `image/` - Local image assets used in the project
